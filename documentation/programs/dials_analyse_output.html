dials.analyse_output¶

Introduction¶

Generate a number of analysis plots from input integrated or indexed reflections.

Examples:

dials.analyse_output indexed.pickle

dials.analyse_output refined.pickle

dials.analyse_output integrated.pickle

Basic parameters¶

output {
  directory = .
}
grid_size = Auto
pixels_per_bin = 10

Full parameter definitions¶

output {
  directory = .
    .help = "The directory to store the results"
    .type = str
}
grid_size = Auto
  .type = ints(size=2)
pixels_per_bin = 10
  .type = int(value_min=1)
centroid_diff_max = None
  .help = "Magnitude in pixels of shifts mapped to the extreme colours in the"
          "heatmap plots centroid_diff_x and centroid_diff_y"
  .type = float
  .expert_level = 1
